# Customer Churn Analysis

## Overview
This project presents an end-to-end Customer Churn Analysis using the Customer Churn dataset. The dataset was cleaned in Microsoft Excel, analyzed using MySQL, and visualized through an interactive Power BI dashboard to identify Revenue impact, and retention opportunities.

## Dataset
- Customer Churn Dataset

## Tools Used
- Microsoft Excel
- MySQL
- Power BI

## Workflow :
1. Cleaned and prepared the dataset using Microsoft Excel.
2. Performed customer churn analysis using SQL queries in MySQL.
3. Built an interactive Power BI dashboard.
4. Generated business insights to support customer retention strategies.

## Key Analysis :
- Customer Churn Rate Analysis
- Customer Retention Analysis
- Revenue Analysis
- Contract-wise Analysis
- Payment Method Analysis
- Monthly Charges Analysis

## Dashboard Features :
- KPI Cards
- Interactive Filters
- Churn Rate Analysis
- Retention Rate Analysis
- Revenue & Churn by Contract
- Revenue Lost by Contract
- Business Insights
